# AI速算

> 公考资料分析速算练习工具 · 策略导向出题 · 弱点精准诊断

## 主要功能

* **🎲 4种练习模式** — 随机出题 / 基础巩固 / 拔高挑战 / 自选练习，覆盖加减法、乘法、除法、复合运算
* **📝 摸底测试** — 20题测出你的速算弱点，首次使用即可获得诊断画像
* **🏥 弱点诊断** — 按模板、陷阱、错误阶段三维度追踪，自动触发弱点报告
* **⏱️ 教练模式** — 40秒提示、60秒跳过、连错警告，模拟真实备考节奏
* **📊 练习历史** — 按日期分组的练习记录，支持错题重练
* **⭐ 速算评级** — 综合正确率、速度、弱点自动评定能力等级

## 特点

* **策略优先出题** — 每道题都有设计好的速算捷径（百化分、凑整、抵消、截位），不是随机数字堆砌
* **陷阱探针选项** — 错误选项匹配具体认知陷阱，选错即知病因
* **纯前端离线可用** — 零依赖，不需要后端，不需要网络，浏览器打开即用
* **移动端优化** — 手机练习体验优先
